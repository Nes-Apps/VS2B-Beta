# VS2B 0.2.0 Beta

This is the first VS2B open-beta release. VS2B is an amateur side project built in spare time, so expect a few rough edges while real people try it on different Windows PCs, phones, tablets, browsers, networks, and games.

## Highlights

- A closer WYSIWYG page-builder preview, including the mobile bottom bar and control-specific visuals.
- Safer save-state and editor-page handling.
- An in-app help guide covering profiles, pages, controls, pairing, and input safety.
- Improved WebSocket reconnection and guarded mobile communication.
- Improved keyboard cleanup and key-release safety.
- Automatic rotating diagnostic logs with an **Export Log** button for bug reports.
- 23 behavioral automated tests covering models, validation, migrations, pairing, diagnostic logging, and log-mode input.

## Please Test

- Whether the PC builder preview matches the phone or tablet layout.
- Dragging and resizing controls with different page sizes and grid modes.
- Saving, unsaved changes, editor-page switching, and app restarts.
- Buttons, holds, toggles, macros, mouse wheels, sliders, energy matrices, and switches.
- Pairing, reconnecting, forgetting devices, and refreshing the mobile page.
- Test Mode, Live Mode, target-application blocking, and **Release All Keys**.

## Important

- This portable Windows x64 build is unsigned, so Windows may show a SmartScreen warning.
- Begin in **Test Mode** before enabling Live Mode.
- Keep VS2B on a trusted local network; do not expose its screen webpage to the public internet.
- Export important profiles before trying a beta build.
- Review exported diagnostic logs for personal paths or network details before posting them publicly.

See the repository README for setup, usage, safety, and bug-reporting instructions.
