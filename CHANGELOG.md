# Changelog

## 0.2.0-beta

- Added a closer WYSIWYG page-builder preview, including the mobile bottom bar and control-specific visuals.
- Separated the editor's selected page from the profile's saved active page to avoid confusing unsaved changes.
- Improved save-state handling so rapid edits use the latest profile data.
- Added an in-app help guide for profiles, pages, controls, pairing, and input safety.
- Improved guarded WebSocket sending and mobile reconnection behavior.
- Improved keyboard toggle cleanup and key-release safety.
- Added 23 behavioral automated tests for models, validation, profile migration, pairing, diagnostic logging, and log-mode input behavior.
- Added an automatic rotating diagnostic log and an Export Log button for beta bug reports.
- Added open-beta instructions and a focused real-device testing checklist.

## 0.1.0-beta.1

- Added safer in-app confirmation dialogs.
- Added configurable hold macro timing.
- Improved input safety and key release behavior.
- Improved PC sidebar organization and collapsible sections.
- Improved mobile page navigation behavior.
- Added paired device management.
- Added slider and mouse wheel control refinements.
